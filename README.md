             SMG Finance Tracker
SMG Finance Tracker is a web-based personal finance application designed to help users organize, categorize, and analyze their M-PESA transactions. The project aims to simplify personal financial management by providing useful insights into income, expenses, spending habits, and savings.


Features
- Import and parse M-PESA SMS messages
- Add manual transactions
- Automatic transaction categorization
- Dashboard showing income, expenses, and balance
- Pie chart of spending categories
- Monthly cash flow chart
- Search transactions
- Filter transactions by month
- Edit and delete transactions
- Export transactions to CSV
- Compound interest calculator
